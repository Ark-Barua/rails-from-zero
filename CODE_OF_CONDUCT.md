# Code of Conduct

## Our Commitment

This project aims to provide a welcoming, beginner-friendly learning environment for everyone interested in Ruby on Rails.

We are committed to creating a respectful and inclusive space regardless of:

- Experience level
- Background
- Identity
- Nationality
- Skill level
- Personal opinions

All participants are expected to follow this Code of Conduct.

---

## Expected Behavior

Participants should:

- Be respectful and constructive
- Use welcoming and inclusive language
- Provide helpful feedback
- Focus on improving learning materials
- Assume good intentions from others
- Help beginners feel comfortable asking questions

---

## Unacceptable Behavior

The following behaviors are not allowed:

- Harassment or discrimination
- Personal attacks or insults
- Offensive or inappropriate language
- Trolling or deliberate disruption
- Spam or self-promotion
- Publishing private information without permission

---

## Contribution Etiquette

When contributing:

- Keep discussions focused on improving content
- Critique ideas, not people
- Explain suggestions clearly
- Be patient with new contributors

Remember: this repository is primarily educational.

---

## Reporting Issues

If you experience or witness unacceptable behavior:

1. Open an Issue labeled **conduct**
2. Describe the situation respectfully
3. Avoid public escalation

Reports will be reviewed and handled appropriately.

---

## Enforcement

Maintainers may:

- Request edits to comments or contributions
- Remove inappropriate content
- Temporarily or permanently restrict participation

Actions are taken to protect a positive learning environment.

---

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Project documentation interactions

---

## Acknowledgment

By participating in this project, you agree to follow this Code of Conduct.
