# Rails From Zero (Free Learning Material)

A beginner-friendly, step-by-step learning path for **Ruby on Rails**, starting from zero.
This repo focuses on **clear explanations**, **practical examples**, and **real project skills**.

## Who is this for?
- Absolute beginners (no Rails knowledge)
- People building a SaaS / web app MVP
- Learners who want a structured path (not random tutorials)

## What you’ll learn
- Ruby fundamentals (the parts Rails uses most)
- MVC architecture (Routes → Controllers → Views)
- ActiveRecord (models, migrations, CRUD)
- Associations (has_many / belongs_to)
- Authentication (Devise)
- Hotwire/Turbo basics
- Background jobs + emails
- Building JSON APIs
- Deployment fundamentals
